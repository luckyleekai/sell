## vue生命周期

sublime text3 编辑器自带的代码格式化可以在 
Edit -> Line -> Reindent
但是sublime text3 没有给它赋予快捷键，所以只需要自己加上快捷键即可

Preference  ->  Key Bindings -user
打开用户快捷键绑定设置添加(比如添加：ctrl + alt + l)
{"keys": ["ctrl+alt+l"], "command": "reindent"}

但是在使用的过程中发现sublime text3自带的编辑器代码格式化能力还是比较弱
必须得要选中要格式的代码才行
